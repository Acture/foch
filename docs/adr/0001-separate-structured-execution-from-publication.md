# Separate structured execution from automatic publication

Foch will run Structured for every parseable Clausewitz-script merge unit. Rollout gates decide only whether a content family's conflict-free results may be published automatically: a known semantic mismatch makes the family review-required, never unsupported, and never triggers a silent Legacy fallback. This accepts additional manual review so foch preserves structured evidence without publishing output known to be unsafe.
